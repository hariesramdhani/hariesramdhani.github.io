Reporter Pathways Analysis Web Tool (Graduation Project)

Reporter Pathways Analysis Web Tool is a web implementation of paper written by Prof. Tunahan Çakır. The tool itself can be accessed through reporterpathways.org. Reporter Pathway Tools allows the user to analysis their transcriptomics data using the method described in the paper. There are four tools available at the moment, Reporter Metabolites Analysis (RMA), Reaction-Centric Reporter Pathway Analysis (RPAr), Metabolite-Centric Reporter Pathways Analysis from Transcriptome Data (Tx-RPAm), Metabolite-Centric Reporter Pathways Analysis from Metabolome Data (Mx-RPAm)

STACKS: Python, R, Django, Django REST Framework, Photoshop CS6, PostgreSQL, RabbitMQ, Celery, Javascript, HTML5, CSS3
